BUILDOGONY

An Evolutionary Game

A simulation game that tests evolutionary change to architectural shapes placed within the population over time.

CONTEXT

MArch + MS EECS Thesis

SOFTWARE + TOOLS

Unity, C#, GeneticSharp

SUPERVISORS

Skylar Tibbits, Stefanie Mueller

Read the full book here

The project explores how architectural space might influence human evolution by simulating it through a virtual game called Buildogeny. Instead of abstract speculation, the game uses evolutionary algorithms and Unity-based gameplay to let digital creatures adapt physically, cognitively, and socially within designed architectural scenarios. Players or researchers can build or test environments, then watch populations evolve in real time, documenting results as evidence for how architecture may shape long-term human futures.

Tests: Maze (2‑wall, 4‑wall, Hampton Court). Findings: 2‑wall → highest speed/lifespan; 4‑wall → most food/larger size; Hampton → lowest lifespan/fewer mates.
